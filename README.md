**Mustache** templates for swift 3.0.

## Installation

- Add `Mustache` to your `Package.swift`

```swift
import PackageDescription

let package = Package(
	dependencies: [
		.Package(url: "https://github.com/Zewo/Mustache.git", majorVersion: 0, minor: 4)
	]
)
```

**Mustache** is a fork of [GRMustache.swift](https://github.com/groue/GRMustache.swift).
